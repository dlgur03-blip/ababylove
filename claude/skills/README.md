# 스킬 폴더

이 폴더는 나중에 **나만의 도구(스킬)**를 추가하는 곳입니다.

기본 제공되는 도구(슬래시 명령)는 `../commands/` 에 있습니다:
- `/기획안` · `/실행` · `/작업보고`

"우리 애는 이런 것도 할 줄 알았으면 좋겠는데?" 싶은 게 생기면,
AI에게 이렇게 말해 보세요:

> "이런 일을 자주 하는데, 이걸 `/이름` 명령으로 만들어줘."

그러면 AI가 이 폴더(또는 `~/.claude/commands/`)에 새 도구를 만들어 줍니다.
(강의의 "교육법 바꾸기 = 에이전트" 부분이 바로 이것입니다.)
