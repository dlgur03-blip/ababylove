# 00. 시작하기 전 준비물 (전제조건)

시작 전에 이것만 확인하세요. 어렵지 않습니다.

## 꼭 필요한 것

### 1. Claude 유료 플랜 (최소 **Claude Pro**)
- 무료로는 이 강의의 핵심 기능(특히 **폰 원격 조종 `/rc`**)이 **작동하지 않습니다.**
- **최소 Claude Pro** 이상(Max/Team/Enterprise 포함)이 필요합니다.
- ⚠️ 회사에서 발급한 "API 키"로는 안 됩니다 — **본인 계정 로그인(Pro 구독)** 이라야 합니다.

### 2. PC를 켜 둘 것 (중요!)
- 폰으로 AI를 조종하는 `/rc` 기능은, **집/사무실 PC에서 AI가 실행 중일 때** 폰이 거기에 "접속"하는 방식입니다.
- 즉 **PC가 꺼져 있으면 폰에서 못 씁니다.** 폰만으로 되는 게 아니에요.
- 밖에서 폰으로 쓰고 싶다면 → **PC를 켜 두고 세션을 실행해 둔 채로** 나가세요.
- 이 차이가 사용 경험을 크게 가릅니다. (켜두면 언제 어디서나 / 꺼두면 PC 앞에서만)

### 3. Claude Code 설치 + 최신 버전
- Claude Code(터미널에서 쓰는 AI)가 깔려 있어야 합니다. (강의 "따라하기"에서 함께 설치)
- 폰 원격(`/rc`)은 **버전 v2.1.51 이상**에서 됩니다. 확인: 터미널에 `claude --version`

### 4. (권장) Python 설치
- 검증 도구(Polyrus)가 파이썬으로 동작합니다. 없으면 설치 안내가 뜹니다.
- 설치: https://www.python.org/downloads/ (또는 AI에게 "파이썬 설치 방법 알려줘")

## 폰으로 조종하기 (`/rc`) — 한눈에

```
1) PC에서 Claude Code 실행 → 대화창에 /rc  (또는 /remote-control) 입력
2) 화면에 뜨는 QR코드를 폰 카메라로 스캔  (또는 세션 주소를 폰 브라우저에 입력)
3) 폰에서 그대로 이어서 대화 → PC의 AI가 실제 작업 수행
```

> 기억: **PC는 켜 둔 채로.** 폰은 "리모컨", 실제 일하는 건 PC 안의 AI입니다.

## 준비 확인 체크
- [ ] Claude Pro(이상) 구독 중
- [ ] `claude --version` 이 v2.1.51 이상
- [ ] 밖에서 쓸 땐 PC를 켜 두기로 이해함
- [ ] (권장) Python 설치됨
