# 03. 배포 기초 — 만든 걸 인터넷에 올리기 (📗 필요할 때)

홈페이지나 간단한 웹을 만들었을 때, 남들이 **주소(URL)로 들어올 수 있게** 올리는 걸 "배포"라고 합니다.

## 개념만

- **내 컴퓨터 안에서만 보이는 것** → 나만 봄.
- **배포(deploy)** → 인터넷 주소가 생겨서 **누구나 그 주소로 접속** 가능.

흔히 쓰는 무료/간편 배포 서비스:
| 서비스 | 쓰임 |
| --- | --- |
| **Vercel** | 웹사이트·웹앱을 몇 분 만에 인터넷에 올림 (가장 쉬움) |
| **GitHub Pages** | 간단한 소개 페이지 올리기 |

## 순서 (개념)
```
만든 웹  →  깃허브에 올림(02번)  →  Vercel에 연결  →  인터넷 주소 생성 🎉
```

## AI에게 이렇게 시키세요
> "이 웹사이트를 Vercel로 배포해서 인터넷 주소를 만들어줘."
> "배포하려면 뭘 준비해야 하는지 하나씩 안내해줘." (계정 연결 등은 손잡고 알려줍니다)

- 계정 로그인이나 연결처럼 **본인이 직접 눌러야 하는 단계**는 AI가 "여기서 이걸 눌러주세요"라고 안내합니다.

## ⚠️ 주의
- 배포하면 **인터넷에 공개**됩니다. 개인정보·비밀번호·비밀키가 페이지에 들어가지 않게 하세요.
- 처음엔 "테스트용"으로 올려보고, 괜찮으면 공유하세요.

## 한 줄 요약
> 배포 = 내 결과물에 **인터넷 주소 붙이기**. "Vercel로 배포해줘"라고 AI에게 시키고,
> 로그인 같은 단계만 안내대로 따라 하면 됩니다.
